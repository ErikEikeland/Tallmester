
# 🎩 Tallmester

Et klasseromsbasert flerspillerspill der elever bruker tall for å løse utfordringer med begrensede siffer.

## 🚀 Kom i gang

1. Installer avhengigheter (krever `npm` og `node`)
2. Start appen med `npm run dev`

## 📁 Mappestruktur

- `src/components`: React-komponenter (GameMaster, PlayerClient, etc.)
- `src/firebase.ts`: Firebase-oppsett
- `firestoreService.ts`: Kommunikasjon med Firestore
